# 簡介

一宗離奇連環命案...

> "啟稟教主，死者有青城派司馬臨、神劍門門主卓亦凡、少林玄空大師、丐幫長老向鼎天，身上均留有一碧綠的怪掌印。"
>
> "師伯死時手中還緊緊握著這珠釵，似是要告訴我們兇手是誰。"
>
> "除了皇帝老子，有誰夠膽跟少林、丐幫過不去！？"

引出一段鮮為人知的塵封記憶...

> "邪神在上，我今日遭此劫難，走頭無路。惟血海深仇未報，死不瞑目！邪神若憐我，賜我力量，助我殺絕仇人，我甘願永生永世，為袮奴僕，絕不食言！"

到底誰才是真兇...

> "今天就是給師父一掌打死，也要弄個明白！！！"
>
> "今日看到這‘降龍神掌’，方知天外有天、人外有人！！！"

隨著時間的推移，真相逐漸浮出水面...
